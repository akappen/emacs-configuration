Emacs 24 Configuration
======================

1.  Clone the repo
    `git clone git://github.com/akappen/emacs-configuration.git`

2.  Symlink ~/.emacs.d to the repo
    `ln -s emacs-configuration ~/.emacs.d`

3.  Install the packages
    `cd ~/.emacs.d`
    `./install-packages`

4.  Use it
    `emacs`
